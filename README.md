# 🌧️ ESP32 Smart Home Control using RainMaker

## 📌 Description

This project is a smart home system built using ESP32 and RainMaker platform to control electrical appliances such as lights and fans remotely.

The system allows users to monitor and control devices via mobile app or cloud, making home automation more efficient and user-friendly.

---

## ⚙️ Features

* Remote control of light and fan using RainMaker
* Wi-Fi connectivity using ESP32
* Real-time device control via mobile app
* Cloud-based control system
* Expandable for more devices

---

## 🛠️ Hardware Used

* ESP32
* Relay Module (for switching devices)
* Light bulb
* Fan
* Jumper wires
* Power supply

---

## 💻 Software Used

* Arduino IDE
* ESP32 Board Package
* RainMaker Library

---

## 🔌 System Architecture

ESP32 connects to Wi-Fi and communicates with RainMaker cloud platform.
Users can control devices (light & fan) through a mobile app, which sends commands to ESP32.

---

## 🚀 How to Run

1. Install Arduino IDE and ESP32 board support
2. Install RainMaker library
3. Connect ESP32 to your computer
4. Upload the code to ESP32
5. Connect relay module to ESP32
6. Configure RainMaker app and pair your device
7. Start controlling your devices remotely
